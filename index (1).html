About This Template

This single file demonstrates semantic landmarks (<header>, <nav>, <main>, <article>, <section>, <aside>, <footer>), a clear heading hierarchy, and accessibility features like labels, a skip link, and meaningful alt text.

Semantic Structure

Simple, adaptive layout that works across mobile and desktop without extra frameworks.
